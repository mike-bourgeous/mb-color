# mb-color

Very small gem with some color functions.  Part of a [video series][0].

## Installation

Add this Git repo to your `Gemfile` then run `bundle install`.

```ruby
gem 'mb-color', git: 'https://github.com/mike-bourgeous/mb-color.git'
```

```bash
bundle install
```
